What it is

Two tools.
One pipeline.

01 — ENGINE
⚙️
Rust Runtime

wgpu rendering, rapier2d physics, hecs ECS, and rodio audio. Tilemap loading for Tiled .tmj files. No web tech — native all the way down.

02 — STUDIO
🖥
Visual Editor

Pure-Rust desktop app built on GPUI (Zed's UI framework). Canvas, asset browser, inspector, scene save/load — no Electron, no JS runtime.

03 — SCRIPTING
📝
Polyglot Logic

Write game logic in Rust, Java, or Python. Scripts open in your editor of choice — VS Code, JetBrains, anything. The studio just gets out of your way.

04 — SCENE
🗂
.ndscene Format

Scenes are plain JSON. Version-control friendly, human-readable, diffable. No binary blobs, no lock-in.

05 — FEEL
🎮
Game Feel Systems

Coyote time, jump buffer, wall-slide. Camera lerp and screen shake. Metroidvania-style room loading. The details that separate a game from a prototype.

06 — NARRATIVE
⚔️
RPG Toolkit

Turn-based battle FSM, branching dialogue with portraits, sprite animation state machine. Built for indie scope without enterprise overhead.
